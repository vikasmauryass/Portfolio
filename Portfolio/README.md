# Vikas Maurya — React Portfolio

Animated, responsive portfolio built with React, Vite, Framer Motion, and
Lucide icons.

## Run locally

```bash
npm install
npm run dev
```

Open the local URL printed by Vite, normally `http://localhost:5173`.

## Production build

```bash
npm run build
npm run preview
```

## Animated water background

The portfolio now uses a lightweight water-ripple background instead of video or
heavy 3D assets. It reacts to mouse movement, creates ripple rings as the cursor
moves, shifts subtly on scroll, and adds mirrored “water reflection” text to the
main sections.
